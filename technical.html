Signal Processing & Analysis

How ActionCensus transforms hand tracking data into precise motor measurements

3D Distance Measurement

Real-time calculation between thumb and index finger tips:

$$d = \sqrt{(x_2-x_1)^2 \cdot W^2 + (y_2-y_1)^2 \cdot H^2 + (z_2-z_1)^2 \cdot W^2 \cdot 0.5}$$

Where $(x_1,y_1,z_1)$ and $(x_2,y_2,z_2)$ are normalized coordinates, $W$ and $H$ are image dimensions.

Hand Size Normalization

Individual variations normalized using wrist-to-MCP distance:

$$d_{norm} = \frac{d_{raw}}{h_{size}} \times 100$$ $$h_{size} = \sqrt{(x_{wrist} - x_{mcp})^2 \cdot W^2 + (y_{wrist} - y_{mcp})^2 \cdot H^2}$$

Ensures consistent measurements across participants with different hand sizes.

Signal Smoothing

Moving average filter reduces noise while preserving dynamics:

$$s[i] = \frac{1}{w} \sum_{k=i-\lfloor w/2 \rfloor}^{i+\lfloor w/2 \rfloor} d[k]$$

Window size $w=3$ balances noise reduction with temporal resolution.

Velocity Analysis

Movement velocity computed via discrete differentiation:

$$v[i] = \frac{s[i] - s[i-1]}{t[i] - t[i-1]}$$

Enables zero-crossing detection for precise contact event identification.

Tap Detection Methods

Four complementary algorithms ensure robust event detection

State Change Detection

Monitors significant closing movements:

$$\text{Event: } \delta d < -3 \text{ AND state} \neq \text{'closing'}$$

Captures rapid finger approaches while filtering gradual movements.

Local Minima Detection

Identifies closest finger positions:

$$s[i-1] > s[i] < s[i+1] \text{ AND } s[i] < \theta_{close}$$

Where $\theta_{close} = 30$ normalized units represents contact threshold.

Velocity Zero-Crossing

Detects movement reversal at contact:

$$v[i-1] < -0.5 \text{ AND } |v[i]| \leq 0.5$$

Captures precise moment of movement reversal at tap contact point.

Rhythmic Prediction

Predicts timing based on established patterns:

$$t_{pred} = t_{last} + \mu_{interval} \pm 0.3\sigma$$

Allows ±30% temporal tolerance for consistent tapping rhythms.

Temporal Filtering

All events filtered with 30ms minimum inter-tap interval to prevent false positives from hand tremor or processing artifacts.

Clinical Metrics

Standardized scores derived from tapping performance using neurological evaluation criteria

Tapping Frequency

$$f = \frac{1000}{\mu_{interval}} \text{ where } \mu_{interval} = \frac{1}{n} \sum_{i=1}^{n}(t_{i+1} - t_i)$$

Primary motor speed metric in Hz. Healthy adults: 4-6 Hz.

Amplitude Analysis

$$A = \max(d_{norm}[t_i : t_{i+1}])$$

Maximum finger separation per tap cycle. Reduced amplitude indicates bradykinesia.

Rhythm Variability

$$CV = \frac{\sigma_{interval}}{\mu_{interval}} \times 100\%$$

Coefficient of variation quantifies timing consistency. Higher CV indicates irregular rhythm.

Amplitude Decrement

$$\text{Decrement} = \frac{A_{initial} - A_{final}}{A_{initial}} \times 100\%$$

Progressive amplitude reduction. >15% may indicate fatigue or motor sequence effects.

Clinical Validation

All metrics validated against Ruff & Parker (1993) normative data and UPDRS motor examination criteria for established clinical correlation.
