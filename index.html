Download & Installation

ActionCensus is a JavaScript library for webcam-based motor assessment. Choose the installation method that works best for your project.

Quick Links

View Library File View on GitHub View on npm

Installation Methods

1CDN (Easiest - Recommended for Beginners)

Include ActionCensus directly in your HTML file. No installation required.

Step 1: Add these script tags to your HTML file:

<!-- Required: MediaPipe Hands --> <script src="https://cdn.jsdelivr.net/npm/@mediapipe/hands/hands.js"></script> <!-- ActionCensus Library --> <script src="https://cdn.jsdelivr.net/npm/actioncensus@latest/dist/actioncensus.min.js"></script>

Step 2: Use ActionCensus in your code:

<script> const assessment = new ActionCensus({ trialDuration: 10000, onComplete: (results) => { console.log('Taps:', results.tapCount); console.log('Frequency:', results.frequency.toFixed(2), 'Hz'); } }); assessment.init().then(() => { assessment.startTrial('right'); }); </script>

2npm (For Node.js Projects)

Install ActionCensus as a package in your JavaScript project.

Step 1: Install via npm:

npm install actioncensus

Step 2: Import and use in your code:

import ActionCensus from 'actioncensus'; const assessment = new ActionCensus({ trialDuration: 10000, onComplete: (results) => { console.log(results); } }); await assessment.init(); await assessment.startTrial('right');
Note: You'll still need to include MediaPipe Hands separately. See the full documentation for details.

3Direct Download

Download the library file and host it yourself.

Step 1: Download the library:

  • actioncensus.min.js (minified, 13.2 KB)
  • actioncensus.js (source code)

Step 2: Include in your HTML:

<script src="https://cdn.jsdelivr.net/npm/@mediapipe/hands/hands.js"></script> <script src="path/to/actioncensus.min.js"></script>

4Clone from GitHub

Get the full source code and examples.

git clone https://github.com/ShadowCoder1/action-census.git cd action-census

The library files are in:

  • src/actioncensus.js - Source code
  • dist/actioncensus.min.js - Built library
  • examples/ - Working examples

Complete Working Example

Here's a complete HTML file you can copy and paste to get started:

<!DOCTYPE html> <html> <head> <title>ActionCensus Test</title> </head> <body> <h1>ActionCensus Motor Assessment</h1> <button onclick="start()">Start Assessment</button> <div id="video-container"></div> <div id="results"></div> <!-- Include libraries --> <script src="https://cdn.jsdelivr.net/npm/@mediapipe/hands/hands.js"></script> <script src="https://cdn.jsdelivr.net/npm/actioncensus@latest/dist/actioncensus.min.js"></script> <script> let assessment; async function init() { assessment = new ActionCensus({ onComplete: (results) => { document.getElementById('results').innerHTML = ` <h2>Results</h2> <p>Taps: ${results.tapCount}</p> <p>Frequency: ${results.frequency.toFixed(2)} Hz</p> `; } }); await assessment.init(); const { canvas } = assessment.getElements(); document.getElementById('video-container').appendChild(canvas); } async function start() { await assessment.startTrial('right'); } init(); </script> </body> </html>
Copy this code, save it as test.html, open it in your browser, and click "Start Assessment"!

Browser Requirements

  • Chrome 80+, Firefox 75+, Safari 14+, or Edge 80+
  • Webcam access
  • HTTPS connection (required for camera access in production)
