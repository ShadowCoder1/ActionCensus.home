ActionCensus: Scalable Webcam-Based Motor Assessment

We introduce ActionCensus, an online motor assessment tool that uses common webcams to infer finger tapping performance in real time. The system self-calibrates through MediaPipe hand tracking and provides clinical-grade motor assessment without specialized equipment.

Try Live Demo View Research

System Architecture

TapSense has two key components: a MediaPipe-based hand detector that tracks finger landmarks, and a motor assessment engine using signal processing to quantify tapping performance.

Real-time Hand Tracking

Utilizes Google's MediaPipe framework for robust hand landmark detection. The system tracks 21 hand landmarks at 30fps, providing precise finger position data for motor analysis.

Signal Processing Pipeline

Advanced algorithms detect finger tapping events through distance analysis, velocity calculations, and multi-method peak detection with temporal filtering for accuracy.

Clinical Metrics

Generates standardized motor assessment scores including tapping frequency, amplitude, rhythm variability, and motor coordination indices validated against clinical measures.

Browser-based Implementation

Runs entirely in web browsers using WebRTC and JavaScript. No server-side processing required, ensuring privacy and reducing latency for real-time assessment.

Cross-platform Compatibility

Compatible with desktop and mobile devices across major operating systems. Optimized performance algorithms adapt to device capabilities automatically.

Data Export & Analysis

Comprehensive data export including raw landmarks, processed signals, and clinical metrics. Integration with research databases and statistical analysis packages.

Research Framework

ActionCensus is designed for rigorous scientific validation with planned studies for population normative data and clinical correlation analysis.

Clinical Research Potential

Motor assessment is critical in neurological evaluation, yet traditional methods require specialized equipment limiting accessibility. ActionCensus addresses this gap by providing a framework for validated motor assessment through standard webcams.

The system is designed to enable research studies that will establish:

  • Correlation with established clinical finger tapping measures
  • Test-retest reliability across sessions and devices
  • Population normative data across age and demographic groups
  • Sensitivity to motor coordination changes over time

Integration & Documentation

ActionCensus can be easily integrated into research platforms, clinical systems, and web applications with minimal setup requirements.

Basic Integration

// Include ActionCensus library <script src="https://actioncensus.org/js/actioncensus.min.js"></script> // Initialize assessment const actionCensus = new ActionCensus({ containerId: 'assessment-container', trialDuration: 10000, // 10 seconds onComplete: (results) => { console.log('Assessment results:', results); // Process clinical metrics exportData(results.clinicalMetrics); } }); // Start motor assessment actionCensus.startAssessment();

Research Protocol

For research applications, we recommend the following standardized protocol to ensure reproducible results and compatibility with existing literature:

  • 6 trials alternating between dominant and non-dominant hands
  • 10-second assessment duration per trial
  • Controlled lighting conditions (indoor lighting, face clearly visible)
  • Standardized instructions and practice trial
  • Rest periods between trials to prevent fatigue
